# Inspecta Reviewer Dashboard - Version 4 UI

A high-density reviewer dashboard for the Inspecta industrial inspection application.

## Features

- **Version 4 UI**: Futuristic 2026 design with electric blue and purple gradients
- Two-pane responsive layout (65% task feed, 35% evidence vault)
- **Expanded task cards** with prominent expand/collapse buttons and default expanded state
- Task cards with severity badges, timestamps, and action buttons
- **Prominent Task Type Icons**: Install (🔧), Repair (🔨), Verify (📋), Clear (🧹)
- **Enhanced Task Status Icons**: Consistent Font Awesome icons across filters and task tiles
- **Advanced Filtering**: Filter tasks by severity, task type, and status with visual icons
- **Highly Visible Video Controls**: Gradient-styled collapse/expand buttons with hover effects
- Video player with seek-to-timestamp functionality
- Collapsible evidence vault for focused task review
- Modern glassmorphism effects and smooth animations

## Tech Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- Font Awesome icons
- Lucide React icons

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the dashboard.

## Build

```bash
npm run build
```

## Mock Data

The dashboard uses mock data for demonstration. Task types and statuses are represented with appropriate icons.
